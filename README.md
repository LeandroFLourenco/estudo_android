## 🎯 Objetivo

Este repositório será utilizado para consolidar conhecimentos em desenvolvimento Android, explorando conceitos como:

- Estrutura de projetos Android
- Layouts XML
- Ciclo de vida de Activities
- Organização de código
- Boas práticas com Gradle

O projeto evoluirá conforme novos estudos forem sendo aplicados.

---

## 🛠 Stack

- Kotlin
- Android SDK
- Gradle (KTS)
- XML
